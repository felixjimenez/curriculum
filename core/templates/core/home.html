Logros recientes con Claude Code

Trabajos que he podido realizar en los últimos meses con la asistencia de Claude Code (IA aplicada al desarrollo de software y a la administración de sistemas). Entre paréntesis indico el perfil profesional que tradicionalmente desempeña cada una de estas labores.

Con la ayuda de Claude Code he podido…

  • Levantar un VPS propio en Hetzner (Debian 13) desde cero, instalar y configurar Docker, Caddy como reverse-proxy con HTTPS automático, organizar /srv/felix para múltiples servicios y dejar el servidor listo para producción. (Administrador de Sistemas Linux / Ingeniero DevOps)
  • Migrar mi DNS a un esquema self-hosted con NSD como master y Buddyns como slave secundario, delegando los nameservers directamente desde NIC.cl y eliminando la dependencia de proveedores externos. (Ingeniero de Redes / Administrador DNS)
  • Registrar y administrar mi dominio felixjimenez.cl directamente en NIC.cl, configurar los subdominios productivos (fotos., manuales., git.) y mantener la zona DNS bajo mi control. (Administrador de Infraestructura)
  • Desplegar este sitio web personal en Django 5.2 en producción, con gestión de media y servicio bajo felixjimenez.cl. (Desarrollador Backend Python / Django)
  • Poner en producción una aplicación Flask para Adderly Construcciones, con base de datos real, integración con Cloudinary para imágenes y panel de administración protegido. (Desarrollador Full-Stack Python)
  • Construir un pipeline completo de Visión por Computadora aplicado a fotografía personal (~3.400 fotos): detección de rostros con OpenCV Haar Cascades, extracción y normalización de caras con dlib (face_recognition), agrupación por similitud con DBSCAN y Chinese Whispers, generación de mosaicos preview y renombrado automático por persona y fecha. (Ingeniero de Visión por Computadora / Data Scientist en IA aplicada)
  • Desarrollar y desplegar una galería web privada (fotos.felixjimenez.cl) con autenticación de administrador, vista pública sin login y herramientas masivas para clasificar personas en categorías (familia, amigos, conocidos…). (Desarrollador Full-Stack)
  • Instalar y administrar mi propio Gitea (git.felixjimenez.cl) como reemplazo de GitHub para repositorios privados, sincronizando dos equipos físicos contra un único origen. (Ingeniero DevOps / SRE)
  • Publicar un sitio de documentación técnica (manuales.felixjimenez.cl) con MkDocs Material — 13 capítulos sobre Visión por Computadora, bitácora de aprendizaje y manuales en PDF — protegido con autenticación básica de Caddy. (Technical Writer / Documentation Engineer)
  • Migrar mi PC principal de Debian 12 a Debian 13 (trixie) sobre disco NVMe, manteniendo Debian 12 en el disco SATA como red de seguridad, y resolver las trampas propias de Python 3.13 (PEP 668, python3.13-venv, setuptools<81, face_recognition_models desde git). (Administrador de Sistemas Linux)
  • Diseñar un flujo de trabajo entre dos equipos físicos (disco SATA y NVMe) sincronizando configuración, memoria de Claude Code y proyectos a través de Gitea, con scripts propios de push/pull al abrir y cerrar sesión. (Ingeniero DevOps)
  • Endurecer la seguridad del VPS y del PC local: configurar sudo NOPASSWD con alcance limitado, llaves SSH dedicadas, certificados TLS automáticos vía Caddy y separación de credenciales por proyecto en archivos .env. (Ingeniero en Seguridad / SysAdmin)
  • Gestionar entornos Python aislados (venv) para cada proyecto cumpliendo PEP 668 de Debian, con dependencias específicas para Flask, Django, Web2py, OpenCV/dlib y MkDocs sin contaminar el sistema. (Desarrollador Python senior)
